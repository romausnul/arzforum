:: Назначение скрипта ::
Данный форум подойдёт тем, кому не нужны все навороты форумов типа phpBB, ExBB, Invision и т.д. Если Вам необходимо начать раскручивать сайт - то этот форум самое то, пока кол-во ваших посетителей невелико. Вы наверно сейчас читаете и думаете: "А когда у меня будет много посетителей, то мне придётся всё потерять и перейти на другой скрипт форума?". Вы не угадали. Как только Вам покажется этот форум недостаточно "навороченным", перейти с сохранением всех данных на WR-Forum Pro версии 2.3 дело 5-и минут! Итак, что есть: высокая скорость работы скрипта, нет глюков при установке, ошибок в коде и отсутствие проблем с форумом. Имеется 9 (девять) скинов и возможность написания своих.
Размер файлов форума: менее 1 Мегабайта.
Пример: форум Лайт на сайте WR-Скрипт.РУ.
:: Возможности ::
1. Чем может похваться форум:

- разработка скрипта ведётся на php 7, проверяется обратная совместимость до php 5.4. Тестируется в браузерах Firefox, Chrom, Opera, Egge;
- Поддержка различных скинов - 9 шт;
- Навигация (быстрый переход по темам);
- Полная встраиваемость в другие скрипты;
- Статистика количества сообщений и тем;
- функции защиты АНТИСПАМ и АНТИФЛУД;
- Наличие регистрации участников;
- Корректное отображение без дополнительных настроек на мобильных телефонах (резиновые таблицы);
- Фильтр-Поиск по названию темы (в выбранном разделе);
- Наличие продвинутого поиска: по всем отправленным сообщениям выбранного участника, поиск сообщений за период, по сообщениям форума;
- Удобная и функциональная "панель администратора";

Подробнее:
1. Универсальная панель администратора с возможностями:
- выбора своего заголовка для всех страниц что формирует рейтинг форума в поисковых системах;
- выбора краткого описания форума для всех страниц отображается в поискиках при поиске в качестве описания;
- выбор количества отображаемых тем / сообщенией на каждой странице;
- возможность перемещения разделов / топиков, а также редактирования и удаления тем / ссобщений.
- в заголовке и титуле страницы отображается строка типа ТЕМА » РАЗДЕЛ » ФОРУМ - что удобно для навигации по форуму и хорошо для поисковиков;
- вствка картинки через тэг [img][/img] отображает любые картинки, расположенные в Интернете. Необходимо указать урл, например: [img]https://ya.ru/ya.gif[/img];
- Антиспам с регулировкой по кол-ву цифр в защитном коде. Возможность включать/выключать этот режим;
- все файлы данных хранятся в одной папке. По умолчанию - "./data". Содержимое папки защищено от взлома файлом .htaccess. По структуре БД идентична WR-Forum версии 2.3 (для перехода к полному форуму достаточно удалить всё, кроме папки "data", закачать все файлы кроме этой же папки из архива WR-Forum 2.3)
- автозамена адреса, например, так: https://www.wr-script.ru
- защищённая от взлома панель администрирования с применением надёжного шифрование MD5 с ключём;
- автозамена тегов на их символьные аналоги;
- автозамена смайликов на картинки (в состав архива входят 11 основных смайликов и 36 дополнительных. Основные: );

Изменения по сравнению с предыдущими версиями:

- Скрипт переведён в кодировку UTF-8;
- Скрипт сохраняет данные в текстовые *.csv файлы. Такие файлы можно редактировать в Excel или Calc. Нагрузку сейчас скрипт держит большую - до 1000 человек в сутки, использование спецбаз MySQL не требуется;
- максимально допустимое количество тем увеличено до 899 и сообщений до 9990 в теме, что позволяет форуму расти и хранить без проблем до МИЛЛИОНА сообщений! Этого достаточно для любого проекта!
- код цифрозащиты "Антиспам" универсален и меняется каждые 24 часа для безопасности и защиты форума от взлома.
- код скрипта адаптирован и структура данных унифицирована для перехода к любой версии WR-форума: MINI, PRO, LUX.
- доработана совместимость с php 7.4. Форум корректно работает с php 5.4 и до 8 версии.

:: Установка и использование ::
1. Создайте папку на Вашем сервере, например "forum-lite". Закачайте в эту папку все файлы и каталоги.
Убедитесь что скрипт работает. Если этого не произошло то, читайте далее.
Выставите ей атрибуты Chmod 755
Закачивать и настраивать атрибуты скрипта можно менеджером "Total Commander".(Файл » Изменить атрибуты).
2. Далее указан состав форума и права доступа, которые необходимо выставить файлам и папкам:
Файлы / описаниеПрава доступа
  • папка "data" содержит файлы данных *.dat и файл настроек;
    		• 666 / 777*
  • всем файлам в папке "data" (кроме .htaccess);
    		• 666 / 777*
  • README.html - этот файл можете удалить после прочтения и установки скрипта;
    		• -
  • index.php - главная страница;
    		• 755
  • admin.php - админпанель. Логин и пароль для входа - admin;
  • antispam.php - функция защиты от Спама "Антиспам";
  • папка "smile" содержит 47 файлов с расширением "gif".
  • *.CSS - файлы - 9 скинов.

    • * - первоначально устанавливайте значение слева от черты, если скрипт не заработал - пытайтесь выставить значение справа. Запустите скрипт ссылкой такого типа: https://Мой_сайт/forum-lite/index.php
    Админпанель запускайте ссылкой вида: https://Мой_сайт/forum-lite/admin.php. Логин и пароль по умолчанию - admin.
    3. Данные права доступа используются на большинстве серверов. В случае, если скрипт не запускается и выводится ошибка "Permise Denied" - скорее всего Вы неверно установили права или что то сделали не так.
    Можете посмотреть статью по настройке прав доступа.
    Если форум не заработал, не бойтесь экспериментировать: изменяйте атрибуты каталогов и файлов.
    Иногда, на "сомнительном" хостинге, форум запускается с атрибутами 777 (вместо 755) и 666 (вместо 644) для файлов и папок соответственно согласно таблице. Либо 777 для всех файлов.
    4. Ваш отзыв или предложения по модернизации скрипта можете оставлять на форуме.
    Если все пункты вы выполнили, а скрипт так и не запустился, ну чтож, заходите на форум в раздел "WR-Script / WR-форум Лайт и Мини" форум по WR-Скриптам и просмотрите темы, возможно решение Вашей проблемы уже есть на форуме. Если нет - задавайте вопросы.
